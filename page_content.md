## My Work

### Digital

- PowderScout
    - PowderScout is a consumer web application that helps powder-chasing skiers and snowboarders chase storms. The app ranks destinations using a scoring algorithm optimized for **fresh snow, affordability, and feasibility**, presenting an actionable recommendation table.
    - URL: https://powderscout.com
    - Technology Stack
        - **Framework**: Next.js 14 (App Router)
        - **Language**: TypeScript
        - **Database**: PostgreSQL with Drizzle ORM
        - **Authentication**: Clerk
        - **Styling**: Tailwind CSS
        - **External APIs**:  Open-Meteo (weather forecasts), Expedia (flights), Google Places (address autocomplete), Google Routes (drive time routing)
- Open Water Insights
    - Open Water Insights is a data visualization tool built for athletes, coaches, and fans in the sport of professional open water swimming. It was created for the purpose of helping people put performances into context, understand how outcomes affect rankings, and compare athletes against each other.
    - URL: https://www.openwaterinsights.com
    - Stack: Python (Plotly, Dash, NetworkX)
